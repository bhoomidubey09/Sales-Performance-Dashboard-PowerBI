# 📊 Sales Performance Dashboard | Power BI

## Overview

This project presents an interactive Power BI dashboard for analyzing sales performance using a relational sales dataset. The dashboard provides insights into sales, profitability, product performance, customer distribution, and operational efficiency.

---

## Dashboard Features

- Total Sales
- Profit Margin
- Profit Margin %
- Total Units Sold
- Top Selling Product
- Sales by Product Line
- Top 5 Products by Sales
- Customer Distribution by Country
- Average Shipping Days by Month
- Quantity Ordered by Year
- Quantity Ordered by Product Line
- Interactive Slicers

---

## Tools Used

- Power BI Desktop
- Power Query
- DAX
- Data Modeling

---

## Dataset

The dashboard uses multiple related tables including:

- Customers
- Orders
- Order Details
- Products
- Product Lines
- Payments
- Employees
- Offices
